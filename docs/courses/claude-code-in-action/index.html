О курсе

Этот курс посвящён Claude Code — консольному AI-ассистенту, который использует языковые модели для выполнения задач разработки.

В курсе разбирается, как Claude Code читает файлы, выполняет команды и изменяет код через свою систему инструментов, а также как управлять контекстом, создавать собственные workflow, расширять Claude Code через hooks и интегрировать его с внешними сервисами.

Что вы изучите

К концу курса вы сможете:

  • Использовать базовые инструменты Claude Code для работы с файлами, командами и анализом кода.
  • Эффективно управлять контекстом через /init, Claude.md и @-упоминания.
  • Управлять ходом сессии с помощью hotkeys и команд.
  • Включать Plan Mode и Thinking Mode для более сложных задач.
  • Создавать собственные команды для автоматизации повторяющихся workflow.
  • Подключать MCP-серверы для расширения возможностей, включая browser automation.
  • Настраивать GitHub-интеграцию для автоматических code review и обработки issue.
  • Писать hooks, которые добавляют дополнительное поведение в Claude Code.

Предварительные требования

  • Базовое знакомство с интерфейсами командной строки.
  • Доступ к Claude Code и API key.

Для кого этот курс

Для инженеров, которые хотят ускорить свой рабочий процесс разработки с помощью AI-ассистента.
