# Claude with the Claude API

Этот курс подробно показывает, как работать с моделями Anthropic через Claude API: от простых запросов до сложных агентов.

Вы узнаете, как формировать запросы, управлять более чем одним ходом диалога, использовать стриминг и структурированные ответы, обрабатывать вложенные инструменты и строить RAG-пайплайны с семантическим поиском и BM25.

В курсе также объясняются особенности интеграции Claude Code и MCP, чтобы связывать Claude с внешними сервисами, а также способы оптимизировать взаимодействие: кэширование подсказок, продвинутое управление контекстом, работа с инструментами/файловой системой и использование генератора команд для автоматизации.
