Работать с Claude Code интереснее, если у вас есть проект, с которым можно работать.

Я подготовил небольшой проект, с которым можно поэкспериментировать в Claude Code. Это то же приложение для генерации UI, которое было показано в предыдущем видео. Примечание: вам не обязательно запускать этот проект. При желании вы всегда можете продолжать остальную часть курса на своей кодовой базе!

Настройка

Для этого проекта требуется немного настройки:

  1. Убедитесь, что у вас локально установлен Node JS. Ссылка на инструкции по установке.
  2. Скачайте zip-файл под названием uigen.zip, прикрепленный к этой лекции, и распакуйте его
  3. В каталоге проекта выполните npm run setup, чтобы установить зависимости и настроить локальную базу данных SQLite
  4. Дополнительно: этот проект использует Claude через Anthropic API для генерации UI-компонентов. Если вы хотите полноценно протестировать приложение, вам нужно предоставить API-ключ для доступа к Anthropic API. Это необязательно. Если API-ключ не указан, приложение все равно будет генерировать некоторый статический фиктивный код. Вот как можно указать API-ключ:
    1. Получите ключ Anthropic API на https://console.anthropic.com/
    2. Поместите свой API-ключ в файл .env.
  5. Запустите проект, выполнив npm run dev
