Пора установить Claude Code локально!

Полные инструкции по установке можно найти здесь: https://code.claude.com/docs/en/quickstart

Кратко, вам нужно сделать следующее:

  1. Установить Claude Code
    1. MacOS (Homebrew): brew install --cask claude-code
    2. MacOS, Linux, WSL: curl -fsSL https://claude.ai/install.sh | bash
    3. Windows CMD: curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
  2. После установки запустите claude в терминале. При первом запуске этой команды вам будет предложено пройти аутентификацию

Если вы используете AWS Bedrock или Google Cloud Vertex, потребуется дополнительная настройка:

  • Отдельные инструкции для AWS Bedrock: https://code.claude.com/docs/en/amazon-bedrock
  • Отдельные инструкции для Google Cloud Vertex: https://code.claude.com/docs/en/google-vertex-ai
