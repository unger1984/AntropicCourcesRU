Вы можете расширять возможности Claude Code, добавляя MCP-серверы (Model Context Protocol). Эти серверы работают либо удалённо, либо локально на вашей машине и дают Claude новые инструменты и возможности, которых у него обычно нет.

Один из самых популярных MCP-серверов — Playwright, который даёт Claude возможность управлять веб-браузером. Это открывает мощные сценарии для рабочих процессов веб-разработки.

Установка MCP-сервера Playwright

Чтобы добавить сервер Playwright в Claude Code, выполните эту команду в терминале, а не внутри Claude Code:

claude mcp add playwright npx @playwright/mcp@latest

Эта команда делает две вещи:

  • Назначает MCP-серверу имя "playwright"
  • Указывает команду, которая запускает сервер локально на вашей машине

Управление разрешениями

Когда вы впервые используете инструменты MCP-сервера, Claude каждый раз будет запрашивать разрешение. Если вас утомляют эти всплывающие запросы, вы можете заранее одобрить сервер, изменив настройки.

Откройте файл .claude/settings.local.json и добавьте сервер в массив allow:

{
  "permissions": {
    "allow": ["mcp__playwright"],
    "deny": []
  }
}

Обратите внимание на двойные подчёркивания в mcp__playwright. Это позволяет Claude использовать инструменты Playwright без запроса разрешения каждый раз.

Практический пример: улучшение генерации компонентов

Вот реальный пример того, как MCP-сервер Playwright может улучшить ваш рабочий процесс разработки. Вместо того чтобы вручную тестировать и подкручивать промпты, вы можете поручить Claude:

  1. Открыть браузер и перейти к вашему приложению
  2. Сгенерировать тестовый компонент
  3. Проанализировать визуальный стиль и качество кода
  4. Обновить промпт генерации на основе увиденного
  5. Проверить улучшенный промпт на новом компоненте

Например, вы можете попросить Claude:

Перейди на localhost:3000, сгенерируй базовый компонент, проверь стили и обнови промпт генерации в @src/lib/prompts/generation.tsx, чтобы в дальнейшем создавать более удачные компоненты.

Claude использует браузерные инструменты, чтобы взаимодействовать с вашим приложением, изучает получившийся результат, а затем изменяет файл с промптом так, чтобы поощрять более оригинальные и креативные дизайны.

Результаты и преимущества

На практике такой подход может заметно улучшить результаты. Вместо типовых фиолетово-синих градиентов и стандартных шаблонов Tailwind Claude может предложить промпты, которые поощряют:

  • Тёплые закатные градиенты (от оранжевого к розовому и фиолетовому)
  • Тематические палитры глубины океана (от бирюзового к изумрудному и циановому)
  • Асимметричный дизайн и перекрывающиеся элементы
  • Творческие интервалы и нестандартные макеты

Главное преимущество в том, что Claude видит реальный визуальный результат, а не только код, и поэтому может принимать гораздо более обоснованные решения по улучшению оформления.

Изучение других MCP-серверов

Playwright — лишь один пример того, что возможно с MCP-серверами. Экосистема включает серверы для:

  • Работы с базами данных
  • Тестирования и мониторинга API
  • Операций с файловой системой
  • Интеграций с облачными сервисами
  • Автоматизации инструментов разработки

Стоит изучить MCP-серверы, которые подходят именно вашим задачам разработки. Они могут превратить Claude из ассистента по коду в полноценного партнёра по разработке, который взаимодействует со всем вашим стеком инструментов.
