## Описание курса

Практический разбор того, как использовать Claude Code для ускорения рабочего процесса разработки.

Курс посвящён `Claude Code` — консольному AI-ассистенту, который использует языковые модели для выполнения задач разработки. В нём разбирается, как Claude Code читает файлы, выполняет команды и изменяет код через свою систему инструментов, а также как управлять контекстом, создавать собственные workflow, расширять Claude Code через hooks и интегрировать его с внешними сервисами.

Здравствуйте и добро пожаловать. Меня зовут Стивен Грайдер.

Я сотрудник технической команды Anthropic.

В этом курсе мы поможем вам уверенно начать работать с Claude Code.

Но прежде чем переходить к техническим деталям, я хочу дать вам короткий обзор того, что мы будем изучать.

Этот курс разделён на четыре части.

Сначала мы разберёмся, что именно собой представляет ассистент для программирования.

Затем рассмотрим Claude Code и поймём, чем он выделяется среди других ассистентов для программирования на рынке.

После этого мы шаг за шагом разберём использование Claude Code на типичном проекте и получим практический опыт.

В завершение мы покажем, как извлечь максимум пользы из Claude Code в ваших собственных проектах.
